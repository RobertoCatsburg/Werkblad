# Zichtbare Buikspieren in 10 Weken

**Startdatum:** 28 juni 2026
**Einddatum:** 5 september 2026

---

## Kernprincipes

Zichtbare buikspieren vereisen twee dingen:
1. **Laag vetpercentage** — mannen ~10-12%, vrouwen ~16-19%
2. **Ontwikkelde buikspieren** — rectus abdominis, obliques, transversus abdominis

Vetverlies is 80% voeding, 20% training. Geen enkel aantal crunches compenseert een slecht eetpatroon.

---

## Fase 1: Fundament (Week 1-3)

### Voeding

| Richtlijn | Details |
|-----------|---------|
| **Calorietekort** | 300-500 kcal onder onderhoudsniveau |
| **Eiwit** | 1.8-2.2g per kg lichaamsgewicht per dag |
| **Koolhydraten** | Focus op complexe koolhydraten (havermout, rijst, zoete aardappel) |
| **Vetten** | 0.8-1g per kg — noten, avocado, olijfolie |
| **Water** | Minimaal 2.5-3 liter per dag |
| **Vermijden** | Suiker, alcohol, bewerkte voeding, frisdrank |

### Voorbeeldeetschema

| Maaltijd | Voorbeeld |
|----------|-----------|
| **Ontbijt (07:00)** | 60g havermout + 1 scoop whey + banaan + kaneel |
| **Snack (10:00)** | 150g kwark + handvol blauwe bessen |
| **Lunch (12:30)** | 150g kipfilet + 200g rijst + groenten + olijfolie |
| **Snack (15:30)** | Rijstwafel + 20g pindakaas + 1 scoop whey shake |
| **Diner (18:30)** | 150g zalm/kipfilet + 250g zoete aardappel + broccoli |
| **Avondsnack (21:00)** | 200g kwark + walnoten (optioneel) |

### Training (4x per week)

**Dag A — Core + Cardio (maandag/donderdag)**

| Oefening | Sets x Reps | Rust |
|----------|-------------|------|
| Plank | 3 x 30-45 sec | 30s |
| Dead bug | 3 x 10 per kant | 30s |
| Crunch | 3 x 15 | 30s |
| Mountain climbers | 3 x 20 | 30s |
| Fietscrunch | 3 x 12 per kant | 30s |
| HIIT cardio (fietsen/roeien) | 15 min | — |

**Dag B — Compound + Core (dinsdag/vrijdag)**

| Oefening | Sets x Reps | Rust |
|----------|-------------|------|
| Squats | 4 x 8-10 | 90s |
| Deadlift | 4 x 6-8 | 90s |
| Overhead press | 3 x 10 | 60s |
| Hanging knee raise | 3 x 10 | 45s |
| Cable woodchop | 3 x 12 per kant | 45s |
| Ab rollout (knieën) | 3 x 8-10 | 45s |

**Woensdag/weekend:** Actief herstel — wandelen (30-45 min), lichte stretching

## Fase 2: Opbouw (Week 4-6)

### Voedingsaanpassingen

- Eiwit verhogen naar 2.2g/kg als nog niet bereikt

## Fase 3: Verfijning (Week 7-10)

### Voedingsaanpassingen
- Calorietekort handhaven op 400-500 kcal
- Natriumintake bewust verlagen (minder opgeblazen gevoel)
- Waterinname verhogen naar 3+ liter
- Groenten bij elke maaltijd (vezels = verzadiging)
- 1 refeed-dag per week behouden

### Training (5-6x per week)

|----------|-------------|------|

| Weighted pull-ups | 4 x 6-8 | 90s |
| Overhead press | 4 x 8 | 60s |

- [ ] Geen suiker/alcohol/bewerkt voedsel

- [ ] Voortgang gelogd

---

## Veelgemaakte Fouten (Vermijden!)

1. **Alleen crunches doen** — Vetverbranding is niet lokaal; compound oefeningen verbranden meer calorieen
2. **Te agressief korten** — Meer dan 500 kcal tekort leidt tot spierverlies
3. **Cardio overdrijven** — HIIT is effectiever dan urenlang joggen
4. **Slaap verwaarlozen** — Cortisol stijgt, vetopslag rond de buik neemt toe
5. **Ongeduld** — Consistentie > perfectie. Eén slechte dag maakt het plan niet kapot
